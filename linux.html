⚠ DATABASE CONSTRAINT BLOCKING SYNC ⚠

Your table virtual_fs has a UNIQUE(name) constraint. This prevents multiple users from having files with the same name, and breaks upsert operations.

STEP 1: Open your Supabase dashboard → SQL Editor (left sidebar)

STEP 2: Paste ALL lines below and click RUN:

ALTER TABLE virtual_fs DROP CONSTRAINT IF EXISTS virtual_fs_name_key;
DROP INDEX IF EXISTS virtual_fs_name_key;
ALTER TABLE virtual_fs ADD COLUMN IF NOT EXISTS owner_id TEXT;
UPDATE virtual_fs SET owner_id = id WHERE owner_id IS NULL;
CREATE UNIQUE INDEX IF NOT EXISTS virtual_fs_owner_name_parent_idx ON virtual_fs(owner_id, name, parent);
